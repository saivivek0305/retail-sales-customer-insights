**Author:** Rapolu Sai Vivek
**Contact:** +91 72070 82048 | cbitrapolu2003@gmail.com

End-to-end data cleaning and RFM segmentation with a Power BI dashboard (add PBIX externally).

## Quick start
1. Install Python dependencies:
   ```bash
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```
2. Place sample `data/sales.csv` and run:
   ```bash
   python src/rfm.py
   ```
3. Open `powerbi/retail_dashboard.pbix` in Power BI Desktop (not included).
